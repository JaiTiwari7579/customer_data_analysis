📊 Customer Shopping Behavior Analysis

📌 Overview

This project focuses on analyzing customer shopping behavior using a complete data analytics workflow. It involves data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and visualization through an interactive dashboard.

The goal is to extract meaningful insights that can help in understanding customer patterns and improving business decisions.

⸻

📂 Dataset
	•	Dataset: Customer Shopping Behavior
	•	Format: CSV
	•	Contains information such as:
	•	Customer ID
	•	Age
	•	Gender
	•	Purchased Items
	•	Category
	•	Purchase Amount
	•	Location

⸻

🛠️ Tools & Technologies
	•	Python (Pandas, Matplotlib)
	•	SQL (PostgreSQL / MySQL / SQL Server)
	•	Power BI (Dashboard creation)
	•	Jupyter Notebook (EDA & analysis)
	•	Gamma (Presentation creation)

⸻

⚙️ Steps Performed

1. Data Loading
	•	Imported dataset using Pandas
	•	Verified structure using .head(), .info(), .describe()

2. Data Cleaning
	•	Handled missing values
	•	Standardized column formats
	•	Removed inconsistencies

3. Exploratory Data Analysis (EDA)
	•	Analyzed:
	•	Gender distribution
	•	Category-wise purchases
	•	Age group segmentation
	•	Created visualizations for better understanding

4. SQL Analysis
	•	Imported data into database
	•	Performed queries like:
	•	Total sales by category
	•	Customer distribution
	•	Location-based insights

5. Dashboard Creation
	•	Built an interactive dashboard in Power BI
	•	Included:
	•	KPIs
	•	Filters (Gender, Category, Location)
	•	Visual trends

6. Reporting
	•	Created a structured report
	•	Built presentation using Gamma

⸻

📊 Dashboard

The Power BI dashboard provides:
	•	Total Sales Overview
	•	Category-wise Analysis
	•	Customer Demographics
	•	Location Insights

⸻

📈 Key Insights
	•	Certain categories dominate customer purchases
	•	Spending patterns vary across age groups
	•	Location plays a major role in purchasing behavior
	•	Gender-based trends are visible in product preferencesi
